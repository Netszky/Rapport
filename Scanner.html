Développement Scanner Chronopost

Description

Mon maitre de stage m'a confié la mission de réaliser un lecteur de Code Barre pour Chronopost. Un SDK développé par la société HoneyWell m'a donc été fourni pour que je réalise cette tâche.
Le but étant de développé avec ce SDK un programme (C) qui sera inséré dans un lecteur de code-barre afin de lire des codes répondant aux critères ChronoPost.

Compétences

  • Langage C

  • Utilisation et compréhension d'un SDK

  • Commande Linux

Ressources Nécessaires

  • Lecteur de Code Barre Honeywell

  • SDK Honeywell

  • Editeur de Code (Langage C)

  • Logiciel Cygwin (Environnement Linux)

1. Utilisation SDK

L'utilisation du SDK nécessite l'installation du logiciel "Cygwin", qui crée un environnement virtuel Linux nécessaire pour la décompression du SDK et son utilisation notamment pour la compilation du fichier C en .moc(à insérer dans le scanner). Le SDK contient des exemples de fichiers commenté, ainsi que des instructions ainsi qu'un guide d'utilisation.

L'image ci-dessus représente la creation et la compilation des fichiers nécessaires avec la commande "make" qui appelle un fichier nommé "MakeFile" créé par HoneyWell dans son SDK qui compile en prenant en compte les fichiers de configuration.
